Computer Science

CS 50

Co-taught with Harvard University, this is a CS class for those with little to no programming experience. This is not designed for CS majors, however, it serves as a good introduction to programming if you don't feel comfortable diving into CPSC 112.

CPSC 112

Introductory programming course for those who are intending to take the CS major and have little to no programming experience.

CPSC 201

Introduction to computer science with those who have some experience with programming. No specific language requirements are needed for this class other than understanding programming logic as a whole. Class is taught in Racket (similar to LISP/Scheme) which may seem unconventional, however, the main focus of this class is on understanding methods of recursion and functional programming rather than object oriented.

CPSC 223

Introduction to algorithms and data structures in C. This class can be difficult when compared to CS
